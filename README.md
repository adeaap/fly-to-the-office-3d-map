# `fly-to-the-office-3d-map`

## Live preview: https://fly-to-the-office-3d-map.vercel.app/

## Tech stack:

- React
- TypeScript
- Mapbox-gl
- Vite

## Run locally

```js
npm install
```

```js
npm run dev
```

#### Note that run locally, you will need a Mapbox API token, set as a `env variable`

.env

```
VITE_MAPBOX_TOKEN=<your-token>
```
